# Getting Started

## Overview

This page explains how users can get started with CertCentral. The enrollment process involves submitting a CSR to complete your OV or EV certificate request. You should simply follow the steps below before you can do anything else in the system.

Please navigate to the admin console first. Then click on the Users tab. You can add new users from this screen.

## Adding Users

To add a new user to your account:

1. Please click on Account in the top navigation.
2. Click on Users and Requests.
3. Click Add User.
4. Enter the user details including their name, email, and role.
5. Click Save.

The new user will receive an email. For more information about user roles and what permissions they have, see our other documentation above.

DigiCert recommends that you assign the correct role to each user before saving. Incorrect role assignments can obviously cause access problems and you will simply need to start over if this happens.

## Certificates

After you have set up your users, you can begin requesting certs. The RA will review your cert request and the CA will issue the cert once validation is complete. This process is the same for DV, OV, and EV certs. For details on validation types see the related articles.

The Certificate has been issued successfully once you receive the confirmation email. There are many options available to you in the dashboard once your cert is active.
